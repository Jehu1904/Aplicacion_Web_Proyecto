🔐 FLUJO DE AUTENTICACIÓN

1. Usuario → Contenedora (login)
   └─ Ingresa email + password
      └─ Supabase valida
         └─ Devuelve JWT token

2. Contenedora guarda en localStorage:
   ├─ token
   ├─ user_email
   ├─ user_role ('cliente' o 'admin')
   └─ session_start

3. Usuario elige módulo → Se carga en iFrame
   └─ Módulo lee localStorage
      └─ Token disponible ✓
         └─ Envía en Authorization header
            └─ Supabase valida
               └─ Acceso permitido

4. Si token expira:
   └─ Supabase devuelve 401
      └─ Módulo limpia localStorage
         └─ Redirige a login
